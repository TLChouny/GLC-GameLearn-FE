User Data Deletion

If you signed in with Facebook and wish to delete your account data from GLC Game Learn, you have two options:

Option A — Self-service request

Send a signed deletion request from Facebook to our endpoint:

POST https://<your-backend-domain>/api/auth/facebook/data-deletion

We will return a confirmation code and a status URL for you to track the deletion.

Option B — Contact support

Email us at support@glc-game-learn.app with the subject "Data Deletion Request" and include the email used to sign in. We will process within 30 days.

See also our Privacy Policy.
